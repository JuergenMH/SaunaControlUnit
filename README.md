# SaunaControlUnit
Control unit for sauna oven. Ensures comfort and safety.
Cheap control units are good for regualation and some basic safetey functions.
This project is designed to add some comfort and safety topics
- Enhanced safety by control internal power lines and oven function
- Enhanced comfort: Acceess via WLAN
- Two relay banks to ensure the system in case of one relay fails
